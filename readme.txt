It is used to make the browser cache the headers for as long as the user indicates.
It improves the experience of visitors.
== Description ==
This plugin adds the option to activate the Browser Cache in the
 "Appearance > Customize" menu.

== Installation ==
1. Unpack the download-package
1. Upload the folder and all folder and files includes this to the
 `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Thats all
